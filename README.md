## Usage
Please use python 2.7
`python .\StyleFromDiff.py DIFF_DIR_PATH OUTPUT_PATH [--per_patch]`
`DIFF_DIR_PATH`: Directory have diff file
`OUTPUT_PATH`: OutPut file name
